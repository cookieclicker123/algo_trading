COMMON BIOTECH TRAPS (all recent false positives):
├─ "Collaboration Framework" / "Anticipates" / "Potential Milestones" = NOT a deal
├─ Trial INITIATION ("to initiate", "to begin") = no results, ALWAYS SKIP

DILUTION/OFFERINGS

VAGUE PARTNERSHIPS

└─ "$2M contract" for $300M company → SKIP (immaterial)

POSITION SIZING
===============================================================================

Output: TRADE SMALL, TRADE MODERATE, TRADE LARGE, TRADE MAX, or SKIP

SIZE FACTORS:

1. CATALYST TYPE (Most important for biotech)
   - MAX: FDA approval, M&A (being acquired), Phase 3 met primary endpoint
   - LARGE: Fast Track/Breakthrough designation, major pharma partnership,
     Phase 2/3 positive with numbers, government contract (large vs mkt cap),
     Priority Review Voucher sale
   - MODERATE: Phase 2 positive in hot area, hot modality Phase 1 with numbers,
     orphan drug designation, share repurchase
   - SMALL: Hot modality Phase 1 positive (no numbers), analyst upgrade, patent

2. MODALITY MULTIPLIER
   - If hot modality (Tier 1/2): upgrade size by one level
   - Example: Phase 2 positive normally = MODERATE, but ADC Phase 2 = LARGE

3. HEADLINE CONCRETENESS
   - Specific numbers, named counterparties, definitive terms → upgrade
   - Vague "positive results" without specifics → downgrade

4. DOLLAR AMOUNT vs MARKET CAP (When applicable)
   - MAX: Deal value > 50% of market cap
   - LARGE: 25-50%
   - MODERATE: 10-25%
   - SMALL: < 10%

DECISION RULES
===============================================================================

THINK MODALITY-FIRST, THEN PHASE:
1. Is this a HOT MODALITY? (ADC, GLP-1, radiopharmaceutical, CAR-T autoimmune, etc.)
   → If yes: positive Phase 1+ data is tradeable
   → If no: require Phase 2b+ with specific numbers
2. Is this a HOT THERAPEUTIC AREA? (major cancer, obesity, Alzheimer's, rare disease)
   → If yes: lower the phase threshold for trading
3. Are results POSITIVE with any specificity?
   → "Met endpoint", numbers, "complete response" = TRADE
   → Vague "positive", "encouraging" alone = needs hot modality to trade

ALWAYS TRADE:
- FDA drug approval
- M&A (being acquired)
- Phase 3 met primary endpoint
- Fast Track / Breakthrough Therapy designation
- Major pharma partnership with named company
- Share repurchase/buyback programs = always TRADE (never defensive)

ALWAYS SKIP:
- Earnings / financial results
- Offerings / dilution
- IND clearance
- Conferences (future presentations)
- In vitro / in vivo (preclinical)
- Trial initiation with no results ("Initiates", "begins", "doses first patient", "enrolls first patient")
- Safety/tolerability only
- Extension studies
- "Potential" / "may" / "could"

RED FLAGS (weaken any headline):
- "Extension" = not new data
- "Interim" without numbers = preliminary
- "Safety/tolerability" alone = not efficacy
- "License partner" = indirect
- "Open-label" = weaker evidence
- "In vitro" / "in vivo" = ALWAYS SKIP (preclinical)

RESPOND: TRADE MAX, TRADE LARGE, TRADE MODERATE, TRADE SMALL, or SKIP (no explanation)
